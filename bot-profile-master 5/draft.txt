                                        Черновой план проекта:
                             - Файл main.py с основной логикой бота
                             - Файл settings.py с токеном, директориями и проч. (техническими функциями бота)
                             - Файл game.py с вопросами-ответами игры
                             - Файл data.py с функциями сохранения-вывода данных о пользователях
                             - Файл saved_data.json с сохранёнными данными пользователей
                             - Файл media.txt со ссылками на облачное хранилище с медиа-файлами
                             - Папка game_media с используемыми медиа-файлами

                                             Принцип работы:
                                Команды:
                                        /start - приветственное сообщение с обращением к пользователю по имени
                                        /about - текстовое сообщение о функционале и командах бота
                                        /game - запускает игру
                                        /endgame - прерывает игру в любой её точке (без сохранения)
                                Алгоритм:
                                        Ответ на сообщение пользователя - сохранение имени пользователя (ключ) в json
                                        Команда /game - начало игры
                                        Ответы на сообщения через кнопки
                                        Бот запоминает ответы пользователя и записывает их во временный словарь
                                        При неправильном вводе предусмотрена конструкция try-except
                                        В ответ на сообщения пользователя выводится соответствующий текст из словаря
                                        После прохождения игры выводится текстовый результат прохождения с медиа-файлом
